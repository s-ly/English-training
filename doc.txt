ОПИСАНИЕ:
Телеграм бот t.me/EngTraining_Bot
Бот для тренировки английских слов.

КАК РАБОТАЕТ:
app.py - главный модуль
func.py - дополнителные функции
log.py - логирование
Texts.py - содержит текстовые данные
MyToken.py - токены для Телеграм
sql.py - для работы с БД

MyToken.zip - архив с токеном
English-training.db - БД
app.log - создаваемый лог, можно удалять
.gitignore - файл игнорирования git
pip_freeze.txt - запись виртуальной среды
doc.txt - этот текст
dict.csv - словарь

env - папка с виртуальной средой
lk - папка, ссылки
